Description:
        In C programming language we use AddressBook to efficiently store and manage personal contact information.It allows
    user to create,search,edit,delete,list all save contacts effeciently.Each contact contains a name,phone number and email 
    address.We validate all contacts with validation.c function by checking wheather saved contacts have name,phone number 
    and email address in correct format and check wheather it stores unique phone number and unique email address.In this project 
    user can select their choice to Create,Search,Edit,Delete or list contacts showed in Address Book Menu.It enables users to 
    perform essential operations such as creating new contacts, searching existing contacts,editing or deleting contact details
    and listing all saved contacts and finally saving the updated information into the created file .After selecting user can 
    able to do any function related to their choice.Thus user create contact then we stores those multiple contacts using an 
    AddressBook structure and saves them to a file named contacts.csv using FILEi/0.
